# DXY-Gold Analysis

This project investigates the relationship between the U.S. Dollar Index and real gold prices using time-series methods.

## Methods
- Rolling Pearson correlation
- ADF test
- Cointegration analysis
- VECM model
- Data visualization

## Tools
Python, pandas, matplotlib, statsmodels

## Output
A 15-page research report and visual analysis of long-term and short-term relationships.
