# DXY-Gold Analysis

This project investigates the relationship between the U.S. Dollar Index and real gold prices using statistical methods.

## Methods
- Rolling Pearson correlation
- ADF test
- Cointegration analysis
- VECM model
- Data visualization

## Tools
Python, pandas, matplotlib, statsmodels

## Output
A 15-page research report and visual analysis of long-term and short-term relationships.

## Key Findings
- Gold prices and the U.S. Dollar Index (DXY) exhibit a persistent but time-varying negative relationship.
- The negative correlation becomes stronger during periods of heightened macroeconomic uncertainty, such as the 2008 financial crisis and the COVID-19 pandemic.
- VAR and impulse-response analysis suggest that spillover effects are generally short-lived, typically dissipating within 1-2 trading days.
